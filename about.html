Acerca del proyecto

Demostración de elementos avanzados de HTML5 y CSS3. Aquí mostramos una tabla con estilos personalizados.

Comparativa de técnicas

Técnica Propósito Compatibilidad
<picture> / srcset Imágenes adaptativas Alta (navegadores modernos)
SVG Gráficos escalables y accesibles Alta
Image map Imagen interactiva con áreas clicables Moderada

Metadatos

  • Favicon en assets/svg/favicon.svg
  • Meta description y meta viewport incluidos

Explicación de los elementos del sitio (requerimiento por requerimiento)

  1. Imágenes y multimedia: Se usan <figure> y <figcaption> en galerías, <picture> y srcset para imágenes adaptativas (ver index.html y resources.html). Además existe un image map interactivo en gallery.html que utiliza <map> y <area>.
  2. SVG y gráficos: El proyecto incluye SVGs propios en assets/svg/ (logo, mapa y nuevos iconos). También se añadió house-plan.svg y se usa como imagen con usemap para áreas clicables.
  3. Animaciones y transiciones: El logo tiene una animación de flotación (keyframes) y hay transiciones en hover para imágenes y tarjetas. Revisa assets/css/styles.css para ver las reglas.
  4. Tablas y formularios: Se añadió una tabla estilizada en about.html (comparativa). Hay múltiples formularios: el formulario de contacto en gallery.html, cuestionario y entrega en resources.html, y el formulario principal con 4 campos. Todos los campos tienen etiquetas accesibles y atributos adecuados.
  5. Metadatos: Cada página incluye <meta> relevantes, y se añadió un favicon SVG assets/svg/favicon.svg.
  6. Maquetación y diseño responsivo: Estructura semántica con <header>, <main>, <footer>. La navegación es coherente entre páginas y el CSS contiene media queries para adaptar el layout (incluyendo el apilado del header desde 950px).
  7. Accesibilidad: Uso de textos alternativos en imágenes, labels en formularios, roles y aria-labels en la navegación. Es recomendable ejecutar una auditoría (Lighthouse o WAVE) para ajustes finos.
